关于本站

  • 这是泽生有史以来做过的最庞大的纯前端项目, 没有之一. 最初完全只是因为想要学习一下 Layui, 不然你说为什么网站标题有个 TE5T 呢?
  • 使用纯 HTML + CSS + JavaScript, 不使用任何开发框架, 不需要任何后端支持. 经过 SSG 改造后虽需要构建, 但搜索引擎应该是会收录了
  • 本站最初的代码跟诗一样, 而且只有至少 2017 年以后的浏览器才能正常加载, 而现在就是为 Android 4 System Webview 设计的, 你甚至可以在 Lynx 等命令行浏览器完整访问本站!
  • 你能通过 HTTP 访问是因为白嫖了 GitHub Pages, Cloudflare 的服务
  • 使用的组件目前只有 Layui (不然从头写太费时间了) 和 lunar.js (就个农历). 顶多加上几个来源不明的图标, 而 Fortune 是从 Litteskin 借来的, 感谢各位
  • 所有代码和文件等均在 GitHub 开源, 分别是 blogtest, blogtest-sitemap (已弃用), gh-netdisk
  • 访问计数器使用 Cloudflare Workers, 这可不是后端, 这叫 Serverless~ 代码懒得放 GitHub, 想要可以电邮我哦~
  • 另外本文使用纯 HTML, 我自写的 Markdown 解析器可能不适合这种复杂的结构
  • 本存储库包括但不限于所有文本内容等直接可见部分使用 知识共享 署名-非商业性使用-相同方式共享 4.0 协议国际版 , 用于包括但不限于文本等内容排版展示等的所有代码使用 MIT License. 本站所有存储库的许可等详细信息可在其根路径下的文件中找到
